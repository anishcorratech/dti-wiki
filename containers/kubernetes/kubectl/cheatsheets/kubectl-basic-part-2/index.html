    Format Kubectl output as rows and columns⚓︎

    Formatting Output as Rows⚓︎

    kubectl get pods -n newrelic -o jsonpath='{range .items[*]}{"pod: "}{.metadata.name}{"\n"}{range .spec.containers[*]}{"\tname: "}{.name}{"\n\timage: "}{.image}{"\n"}{end}'
